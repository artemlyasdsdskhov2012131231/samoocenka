# 📦 APK BUILDER - Полный набор инструментов

Приложение **"Самооценка состояния"** теперь готово к сборке в APK для Android!

## 📂 Структура файлов сборки

```
d:\анкета состояния 2\
├── 📄 QUICK_START_APK.md          ← НАЧНИТЕ ОТСЮДА!
├── 🔧 CHECK_ENVIRONMENT.bat       ← Проверка окружения
├── 🔨 BUILD_APK.bat               ← Главный скрипт сборки
├── 🔨 BUILD_APK.ps1               ← Альтернатива (PowerShell)
├── ⚙️  buildozer.spec             ← Конфигурация (основная)
├── ⚙️  buildozer_clean.spec       ← Конфигурация (альтернативная)
├── 📖 BUILD_APK_GUIDE.md          ← Полная инструкция
├── ☕ INSTALL_JAVA.md             ← Установка Java
├── 📋 APK_BUILD_README.md         ← Этот файл
│
├── app.py                         (исходный код)
├── storage.py                     (управление хранилищем)
├── requirements.txt               (зависимости Python)
└── assets/                        (иконки и ресурсы)
```

## 🚀 БЫСТРЫЙ СТАРТ (3 шага)

### 1️⃣ Проверка подготовки
```bash
CHECK_ENVIRONMENT.bat
```
Проверит наличие Java, Android SDK, Python, Buildozer

### 2️⃣ Установка недостающих компонентов
Формируется если проверка выявит проблемы:
- **INSTALL_JAVA.md** - установка Java Development Kit
- **BUILD_APK_GUIDE.md** - полное руководство для каждого инструмента

### 3️⃣ Сборка APK
```bash
BUILD_APK.bat
```
Выберите:
- **Опция 1** - Debug APK (для тестирования)
- **Опция 2** - Release APK (для публикации)

## 📖 Полная документация

| Документ | Для кого | Когда читать |
|----------|----------|-------------|
| **QUICK_START_APK.md** | Все разработчики | В первую очередь |
| **CHECK_ENVIRONMENT.bat** | Все | Перед сборкой |
| **BUILD_APK_GUIDE.md** | Новички | Если нужна подробная информация |
| **INSTALL_JAVA.md** | Если нет Java | Установка JDK |
| **buildozer.spec** | Продвинутые пользователи | Для кастомизации сборки |

## 🎯 Результат сборки

После успешной сборки APK находится в:
```
bin/samoocenka-1.0.0-debug.apk     (для тестирования)
bin/samoocenka-1.0.0-release.apk   (для публикации)
```

## ⏱️ Время сборки

- **Первый раз**: 15-30 минут (скачивание зависимостей)
- **Следующие разы**: 5-15 минут

## 💻 Системные требования

- Windows 10/11 (или Linux/Mac)
- Python 3.9+
- Java Development Kit (JDK) 11+
- Android SDK (API 31+)
- Android NDK
- ~5 ГБ свободного места на диске
- ~1.5 ГБ RAM

## 🔧 Конфигурационные файлы

### buildozer.spec
Основной файл конфигурации для сборки. Содержит:
- Название и версию приложения
- Необходимые зависимости Python
- Разрешения Android
- Целевую версию API
- Архитектуры для сборки (arm64-v8a, armeabi-v7a)

Можно редактировать для кастомизации сборки.

### buildozer_clean.spec
Альтернативная, более чистая конфигурация без рекомендуемых параметров.

## 📱 Установка на телефон

### Способ 1: Через USB (рекомендуется)

```bash
# Включите "Отладка USB" на телефоне
# Подключите USB кабелем
# Установите APK:
adb install bin\samoocenka-1.0.0-debug.apk
```

### Способ 2: Вручную

1. Передайте APK на телефон (по почте, Bluetooth, облако)
2. Откройте файловый менеджер
3. Найдите APK файл
4. Нажмите для установки
5. Разрешите установку из неизвестных источников (если просит)

## 🐛 Решение проблем

### Проблема: "Java not found"
**Решение**: Смотрите `INSTALL_JAVA.md`

### Проблема: "Android SDK not found"
**Решение**: `BUILD_APK_GUIDE.md` раздел "Установка Android SDK"

### Проблема: Очень долгая сборка
**Это нормально** при первом запуске.

### Проблема: "Out of memory"
**Решение**: Закройте другие приложения, особенно браузеры

### Проблема: Приложение вывалится на телефоне
**Решение**: 
1. Проверьте версию Android >= 5.0
2. Смотрите логи: `adb logcat | grep samoocenka`
