# ☕ Установка Java Development Kit (JDK)

JDK требуется для сборки Android приложений.

## 🪟 Windows 10/11

### Способ 1: Через Eclipse Temurin (Рекомендуется)

1. Откройте https://adoptium.net/
2. Скачайте **OpenJDK 17 LTS** (Windows Installer)
3. Запустите установщик
4. При установке отметьте "Add to PATH" ✅
5. Закройте все PowerShell окна и откройте новое

### Способ 2: Через Oracle

1. Откройте https://www.oracle.com/java/technologies/downloads/
2. Скачайте **JDK 17** (Windows x64 Installer)
3. Запустите установщик `jdk-17_windows-x64_bin.exe`
4. Установите в папку по умолчанию

### Способ 3: Через Chocolatey (если установлен)

```bash
choco install openjdk17
```

## ✅ Проверка установки

Откройте PowerShell и проверьте:

```bash
java -version
```

Должны увидеть что-то вроде:
```
openjdk version "17.0.6" 2023-01-17
OpenJDK Runtime Environment (Temurin) (build 17.0.6+10)
```

Если ошибка - перезагрузитесь или установите переменную окружения вручную.

## 🔧 Установка переменной окружения (если нужно)

Если `java -version` не работает после установки:

### На Windows 10/11:

1. Откройте **Параметры** → **Система** → **О системе** → **Параметры среди** (внизу)
2. Нажмите **Переменные среды**
3. Нажмите **Новая** (в разделе "Переменные пользователя")
4. Создайте переменную:
   - **Имя**: `JAVA_HOME`
   - **Значение**: `C:\Program Files\Eclipse Adoptium\jdk-17.0.6\` (или где установлен ваш JDK)
5. Нажмите OK

Затем добавьте Java в PATH:
1. В том же окне найдите переменную **Path**
2. Нажмите Edit
3. Нажмите New
4. Добавьте: `%JAVA_HOME%\bin`
5. Нажмите OK несколько раз

Перезагрузитесь и проверьте: `java -version`

## 🎯 Успех!

Если `java -version` показывает версию - JDK установлен правильно!

Продолжайте со следующего шага в `BUILD_APK_GUIDE.md`
