# Запуск на другом ПК (Portable)

## Быстрый старт

1. **Скопируй папку `234` с флешки на компьютер** (любой диск)
2. **Запусти `START_APP.bat`** - двойной клик
3. **Откроется браузер** с приложением на http://localhost:8550

## На ЧТО обратить внимание:

✅ Python 3.11+ должен быть установлен на ПК  
✅ Первый запуск может быть медленнее (установка зависимостей)  
✅ Все данные сохраняются в папке `results/`  
✅ На диске D флешки автоматически создаётся папка `234` с приложением

## Переносимость

- ✅ Работает на Windows 10, 11
- ✅ Не требует интернета (offline mode)
- ✅ Не требует установки (portable version)
- ✅ Все зависимости скачиваются автоматически

## Если что-то не работает

Запусти в командной строке (PowerShell):
```
cd d:\234
python run_offline.py
```

Проверь ошибки в консоли.
