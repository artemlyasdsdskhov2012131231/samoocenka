# 🚀 БЫСТРЫЙ СТАРТ: Сборка APK

## ⚡ За 3 шага

### Шаг 1️⃣: Проверка окружения
```bash
CHECK_ENVIRONMENT.bat
```
Проверит все необходимые инструменты (Java, Android SDK, Buildozer и т.д.)

### Шаг 2️⃣: Установить недостающие компоненты (если нужно)

Если что-то не установлено, следуйте инструкциям в:
```
BUILD_APK_GUIDE.md
```

### Шаг 3️⃣: Собрать APK
```bash
BUILD_APK.bat
```

Выберите:
- **1** - Debug APK (для тестирования на телефоне)
- **2** - Release APK (для публикации)

## ⏱️ Время сборки

- **Первый запуск**: 15-30 минут (скачивание зависимостей)
- **Следующие сборки**: 5-15 минут

## 📱 Установка на телефон

**Способ 1: Через adb (рекомендуется)**
```bash
adb install bin\samoocenka-1.0.0-debug.apk
```

**Способ 2: Вручную**
1. Передайте APK на телефон
2. Откройте в файловом менеджере
3. Нажмите установить

## 🐛 Если что-то не работает

1. Запустите `CHECK_ENVIRONMENT.bat` - проверит все компоненты
2. Смотрите `BUILD_APK_GUIDE.md` - полная инструкция для каждого шага
3. Проверьте лог в папке `.buildozer/`

## 📋 Файлы в проекте

| Файл | Назначение |
|------|-----------|
| `CHECK_ENVIRONMENT.bat` | Проверка подготовки системы |
| `BUILD_APK.bat` | Сборка APK (главный скрипт) |
| `BUILD_APK.ps1` | Альтернативный скрипт (PowerShell) |
| `buildozer.spec` | Конфигурация сборки |
| `buildozer_clean.spec` | Альтернативная конфигурация |
| `BUILD_APK_GUIDE.md` | Полная инструкция с решением проблем |

## 💡 Советы

- ✅ Убедитесь что `app.py` находится в корне проекта
- ✅ На первую сборку запаситесь терпением (30+ минут)
- ✅ Последующие сборки будут намного быстрее
- ✅ Для тестирования используйте Debug APK
- ✅ Для публикации используйте Release APK с подписью

## 🎯 Готово!

После успешной сборки APK будет в папке:
```
bin/samoocenka-1.0.0-debug.apk
```

Установите на телефон и тестируйте!
